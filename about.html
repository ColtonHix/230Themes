    By Brig Lowell

"The Swiss Army Chainsaw"

Flexible, Powerful, Ugly

Perl or also referred to as the “Swiss Army Chainsaw of Scripting Languages,” was originally developed by Larry Wall on December 18, 1987 as a general-purpose Unix scripting language to make report processing easier. The name Perl was originally derived from “pearl”. Wall looked over all 3-4 letter words in the dictionary when attempting to come up with a name. He thought of using his wife Gloria’s name but landed on pearl later shortened to perl.

Perl has had many versions and revisions over the years. Each year bringing new features, functionality, and efficiency. The following is a brief look over the versions released leading up to the version Perl 5 used today.

  • Perl 1.0 1987, Released to comp.sources.misc newsgroup
  • Perl 2 1988, Better regular expressions added
  • Perl 3 1989, Support for binary data streams
  • Perl 4 1991, first documented book released “Programming Perl” aka “The Camel Book” (originally relied on lengthy man page)
  • Perl 5.000 released October 17 1994
  • Perl 5.001 March 13 1995, On October 26, 1995, the Comprehensive Perl Archive Network (CPAN) was established as a repository for the Perl language and Perl modules; as of May 2017, it carries over 185,178 modules in 35,190 distributions, written by more than 13,071 authors, and is mirrored worldwide at more than 245 locations.
  • Perl 5.002 and 5.003 1996, added prototype features
  • Perl 5.004 May 15 1997, UNIVERSAL package giving base object all classes derive from. CGI.pm module developed gaining popularity as a CGI scripting language. Plus added Operating System support for Windows and others.
  • Perl 5.005 July 22, 1998 Regex engine added and backend hooks added.

Perl 5 was one of, if not, the biggest updates with an almost complete rewrite of the interpreter. Additionally it added objects, lexical variables, references, and modules. Perl refers to Perl 5 today, as redesigned Perl 6 was renamed to Raku in October 2019. Perl 5 has had a number of updates over the years enhancing core items, adding features, and extending compatibility to keep the language up to date with modern devices and software. The current stable version available to download on Perl.org, as of Nov, 2019, is 5.30.1 which released 5-22-2019.

Where is Perl Used?

Perl has chiefly been used to write Common Gateway Interface (CGI) scripts. Some well known examples written in perl are:

  • Craigslist
  • DuckDuckGo
  • IMDb
  • Bugzilla

Perl is also referred to as "duct tape" for its ability to glue/tie systems together that were not designed specifically to interpolate and mapping one "raw" type of data format to another more valuable format. This makes Perl popular for system administrators, bioinformatics, financial analytics and more.
